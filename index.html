1.¿Qué es HTML?

Para definirlo de forma sencilla, podriamos decir que HTML es utilizado para crear todas las páginas de internet o que es el lengueaje con la que se escribe la mayoria de las páginas web

2.¿Qué es CSS?

Es un leguaje de estilos en cascada, se de denomina estilos en cascada porque se aplica de arriba a abajo, CSS es el lenguaje con el cual vamos a dar estilos visuales y estéticos a la estructura HMTL

3.¿Qué es Javascript?

Con HTML y CSS ya podemos tener estructura, organizada y visualmente bonita la página web, si queremos alterar documentos de forma dinámica, personalizar eventos o responder al usuario tenemos el lenguaje de programación Javascript, por tanto Javascript es un leguaje de programación usado para procesar información y manipular documentos.
